# Advisors

An app to make information and access to advisors more convenient. Features tabs for lawyers, consultants, teachers, and AI. Advisors can post themselves and charge a low sum for quick chats or texting services.

This is a [Next.js](https://nextjs.org) project bootstrapped with [`create-next-app`](https://nextjs.org/docs/app/api-reference/cli/create-next-app).

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the page by modifying `app/page.tsx`. The page auto-updates as you edit the file.

## Features

- Tabs for different advisor categories: Lawyers, Consultants, Teachers, AI
- User authentication and profiles
- Payment processing for services
- Real-time chat functionality

## Tech Stack

- Next.js 16
- TypeScript
- Tailwind CSS
- ESLint
